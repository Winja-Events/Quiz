1. **Can You Answer This?**
   To extract absolute URLs from source code, which of the following regular expressions **cannot** be used?

    ```bash
    http(s)*://[^”]*
    http[s]?://[^”’]+
    [\w]+[s]?[/:]{3}[^”’]+
    [\w]+[s]?[^”’]+
    [\w]+[s]?://[^”’]+
    ```

    ![Quiz1](image/Quiz1.png)

    **Answer:** #4 <br/>
